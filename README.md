# kot_brodskogo

## Requirements

 * Python `.python-version`
 * Node.js `.nvmrc`
 * Yarn `.yvmrc`

## Install

1. Initialize project

```sh
./install.sh
```

3. Run server

```
python manage.py runserver
```

4. Open server URL in browser
5. Navigate to `/admin` and login with `admin` username with `admin` password

## Update

```sh
./update.sh
```
